## Project Title

Side Quest Week 4: Mini Mines

---

## Description

A Minesweeper-inspired-and-simplified version

---

## Interaction Instructions

> Click Start to begin the game. The board is made up of hidden tiles.
> Click tiles one at a time:

    Safe tile → turns green
    Mine tile → turns red and the game ends

> There are no numbers or hints—players must rely on memory and observation.
> To win, reveal all safe tiles without clicking on a mine.
